# Ảnh chụp: Admin Panel

> **Đường dẫn**: `/admin`

## Mô tả nội dung ảnh

Trang Super Admin dành cho tài khoản có `role = 'admin'`, bao gồm:

- **Dashboard Admin (`/admin`)**
  - Cards thống kê: Tổng số gia đình, nhánh họ, bot đang hoạt động
  - Biểu đồ thống kê request AI theo ngày (7 ngày / 30 ngày)
  - Danh sách gia đình gần đây hoạt động

- **Sidebar điều hướng**
  - 🏠 Tổng quan
  - 🤖 Cài đặt AI Platform (`/admin/ai-settings`)
  - 💳 Quản lý Thuê Bao (`/admin/subscriptions`)
