# Ảnh chụp: AI Settings

> **Đường dẫn**: `/admin/ai-settings`

## Mô tả nội dung ảnh

Trang cài đặt AI platform-wide cho Super Admin, bao gồm:

- **Cài đặt nhà cung cấp AI**
  - Dropdown: OpenAI / Anthropic / OpenRouter / LiteLLM
  - Ô nhập API Key dùng chung cho toàn platform
  - Ô nhập Base URL (cho OpenRouter / LiteLLM self-hosted)
  - Ô nhập tên model mặc định

- **Thông tin sử dụng**
  - Tổng số request AI trong tháng hiện tại
  - Phân bổ request theo gia đình (top 10)
  - Cảnh báo khi sắp đạt giới hạn quota

- **Ghi chú BYOK**
  - Gia đình dùng BYOK (key riêng) sẽ không tiêu thụ quota platform
  - Gia đình không có key riêng sẽ dùng key platform này
