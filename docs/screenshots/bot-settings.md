# Ảnh chụp: Bot Settings

> **Đường dẫn**: Dashboard → Cài đặt → Nhánh họ → [tên nhánh] → Bot

## Mô tả nội dung ảnh

Màn hình cài đặt bot cho từng nhánh họ, bao gồm:

- **Tab Telegram Bot**
  - Ô nhập Bot Token (từ @BotFather)
  - Nút **Đăng ký Webhook** — tự động cấu hình webhook URL
  - Trạng thái webhook: ✅ Đã kết nối / ❌ Chưa cấu hình

- **Tab Zalo OA**
  - Ô nhập Access Token Zalo Official Account
  - Nút **Lưu cài đặt**

- **Tab AI**
  - Dropdown chọn nhà cung cấp: OpenAI / Anthropic / OpenRouter / LiteLLM
  - Ô nhập tên model (vd: gpt-4o, claude-3-5-sonnet)
  - Ô nhập API Key riêng (BYOK) — tùy chọn
  - Textarea System Prompt — tùy chỉnh nhân cách bot
